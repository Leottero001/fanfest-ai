# Supabase - FanFest AI

Este directorio contiene los archivos correspondientes al backend de Supabase:

## Estructura
* `/migrations`: Contiene los scripts SQL de migración de base de datos PostgreSQL.
* `/functions`: Contiene el código de las Supabase Edge Functions (Deno / TypeScript) para lógica de backend serverless.

## Inicialización Local (Opcional)
Para trabajar localmente con Supabase CLI:
1. Instala Supabase CLI.
2. Ejecuta `supabase init`.
